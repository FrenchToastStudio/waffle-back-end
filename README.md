# Run

1. ```
   python3 -m venv venv
   ```

2. ```
   . venv/bin/activate
   ```

3. ```
   pip install Flask
   ```

4.  `Run Flask`

5. ```
   export FLASK_APP=flaskr
   ```

6. ```
   export FLASK_ENV=development
   
   ```

7. ```
   flask run
   ```
